# 投稿データの通信について

## 必要なもの

* アイコン画像
* ユーザー名
* フォロー中のユーザーの数
* イチオシ料理
* BIO
* 投稿数
* 誕生日(年/月/日)
* 性別
* 国、都道府県
* SNS垢

---

## Post テーブル

| name | type | detail |
| --- | --- | --- |
| name | string | ユーザーの表示名 |
| created_at | date | 作成時刻 |
| updated_at | date | 更新時刻 |
| biography | text | BIO |
| birth | date | 誕生日 |
| country | string | 国 |
| prefecture | string | 都道府県 |
| icon_address | string | アイコン画像のアドレス |
